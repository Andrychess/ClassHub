# Changelog

## 1.1.0

### Классы и роли

- Несколько классов у преподавателя: папка материалов и whitelist программ по **имени ярлыка**
- Ученик выбирает класс в сети; настройки класса подтягиваются с ПК преподавателя
- Запоминание роли (ученик / преподаватель) между запусками
- Серые плитки «Не установлена» для программ класса, которых нет на этом ПК

### Интерфейс

- Навигация: **Урок** (рабочая страница, трансляции) и **Управление** (настройки класса, сеть)
- Обновлённая панель навигации, страницы настроек и сети в двух колонках
- Вкладка **Трансляции** с полноэкранным режимом просмотра экранов
- Поиск программ при настройке класса

### Надёжность

- Обработка `STOP_SOURCE` в discovery — корректное снятие статуса «источник»
- TTL устройств в списке peers (offline исчезают через ~60 с)
- Чат привязан к `classId` класса
- «Сменить роль» сбрасывает вход ученика в класс (`leaveClass`)
- Подтверждения при удалении класса и смене роли
- Debounce обновления списка классов при смене peers
